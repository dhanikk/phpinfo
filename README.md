## **Features**  
- **Comprehensive PHP Info** ‚Äì Displays PHP version, extensions, configuration, and modules.  
- **Debugging & Troubleshooting** ‚Äì Identify configuration issues quickly.  
- **Performance Optimization** ‚Äì Gather insights for tuning PHP performance.  
- **PHP Directives Overview** ‚Äì Shows key settings like memory limit and execution time.  
- **Search Functionality** ‚Äì Find specific PHP settings easily.  
- **Extension & Module Check** ‚Äì Lists enabled and disabled PHP extensions.

## **Installation**  
To install the package Open the terminal and run the following command:  
<pre><code class="language-bash">composer require itpathsolutions/phpinfo</code></pre>   

## **Commands**

## **Accessing the Plugin**
